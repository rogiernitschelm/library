## Cargo

Example package:

```rust
[package]

name = "hello_world"
version = "0.0.1"
authors = [ "Your name <you@example.com>" ]
```

Run:

```bash
# Inside the directory of the package

cargo run
```

Release:

```bash
cargo build --release
```

Creating a project:

```bash
cargo new hello_world --bin
```
