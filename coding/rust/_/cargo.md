## Cargo

Example package:

```rust
[package]

name = "hello_world"
version = "0.0.1"
authors = [ "Your name <you@example.com>" ]
```

Build:

```bash
cargo build
cargo build --release
```

Build and run:

```bash
cargo run
```

Creating a project:

```bash
cargo new project --bin
```

Creating a library:

```bash
cargo new library
```

Update:

```bash
cargo update
```

Open documentation:

```bash
cargo doc --open
```
